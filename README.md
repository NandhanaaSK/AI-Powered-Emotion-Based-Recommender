# AI-Powered Emotion-Based Recommender System

AI-Powered Emotion-Based Recommender System is an intelligent web application that detects users' emotions through facial expression recognition and provides personalized recommendations, activities, and interactive games to enhance user engagement and emotional well-being.

## ✨ Features

📷 Real-time facial emotion detection using webcam

🤖 Deep Learning-based emotion recognition using CNN

🎭 Detects multiple emotions:

* Angry
* Disgust
* Fear
* Happy
* Sad
* Surprise
* Neutral

🎮 Interactive emotion-based games and activities

💡 Personalized recommendations based on detected emotions

🌐 User-friendly and responsive web interface

⚡ Real-time emotion prediction and recommendation system

## 🚀 How It Works

1. User accesses the web application.
2. The webcam captures the user's facial image.
3. OpenCV detects the face from the image.
4. The trained CNN model predicts the user's emotion.
5. The system generates personalized recommendations and activities.
6. Interactive games are displayed based on the detected emotional state.
